
# 엑셀 텍스트 변경 감지기 (A~K열)

두 개의 엑셀 파일을 업로드하여 **A열~K열** 범위에서 **문자값이 바뀐 셀**만 찾아서
`K열 3행 'S'에서 'M'(으)로 변경됐음` 같은 한국어 메시지로 보여주는 스트림릿(Web) 앱입니다.
셀의 **배경색(채우기)만 바뀐 경우**는 기본적으로 *변경으로 간주하지 않고 제외*합니다.

## 기능
- 업로드한 두 Excel 파일의 원하는 시트를 선택하여 비교
- A~K 열 범위 안에서 **문자값(text)** 변경만 검출
- (옵션) 색이 칠해진 셀은 변경 검출에서 제외
- (옵션) 앞뒤 공백 무시, 대소문자 구분/무시 선택
- 결과를 표로 확인하고 CSV/Excel로 다운로드

## 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub 업로드 및 Streamlit Community Cloud 배포
1. 이 리포지토리(또는 파일들)를 GitHub에 업로드합니다.
2. [Streamlit Community Cloud](https://share.streamlit.io/)에 접속해 새 앱을 생성하고,
   - **Repository**: 업로드한 깃허브 리포지토리
   - **Branch**: main (또는 선택한 브랜치)
   - **Main file path**: `app.py`
3. 배포를 시작하면 곧 앱이 생성됩니다.

## 사용 팁
- 비교 기준은 **행 번호** 기준입니다. (예: A3 vs A3)
- 값 비교는 기본적으로 **문자형**에 한합니다. 숫자/날짜가 텍스트로 저장된 경우에도 비교가 가능합니다.
- "색이 칠해진 셀 제외" 옵션을 켜면, 두 파일 중 어느 한쪽이라도 셀에 채우기가 되어 있으면 해당 셀은 무시됩니다.
- 더 확장하여 범위를 바꾸거나 키 매칭(예: ID 열 기준)으로 비교하고 싶다면 로직을 쉽게 확장할 수 있습니다.
